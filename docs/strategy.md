# 戦略 — Open Ground Coffee

## 売りたいもの

**スペシャリティコーヒー豆の焙煎品**。自前EC（構築中）から直販。
カフェ卸も並行して継続（卸経由で知ってもらう入口にも機能）。

## ペルソナ（優先順）

### Primary: 家で淹れる趣味層
- 既にコーヒー器具（ドリッパー、ミル）を所有
- 1 袋 ¥1,500〜¥3,000 に抵抗がない
- 産地・精製方法・ロースト日などの情報で買う豆を決める
- 失敗した淹れ方のリカバリに興味がある
- **この層を取れなければ事業成立しない**

### Secondary: 開業したての小規模カフェ店主
- 卸先として直接的な収益
- 1 店舗が決まると月5〜15kg の安定需要
- SNS は広告ではなく「焙煎所を知ってもらう信頼シグナル」として機能

### Tertiary: ギフト需要
- おしゃれなビン/パッケージが意思決定に効く
- 父の日、誕生日、開業祝いなど季節キャンペーンで拾う

## 差別化ポイント

1. **スペシャリティにこだわる知識量** — 焙煎士が一次情報を発信
2. **ビン・パッケージの意匠** — 贈答にそのまま使える
3. **SNS で焙煎士の顔が見える** — ECの店主が誰かわかる安心感

## コンテンツ柱（推奨比率）

| 柱 | 比率 | ねらい |
|---|---|---|
| ①焙煎士の一次情報 | 30% | 競合少、差別化が最強 |
| ②産地×味を具体語で描写 | 25% | 買う人の「選ぶ基準」を育てる |
| ③抽出のこだわり | 20% | 既存客の粘着、新規の入口 |
| ④ビン/パッケージ見せる | 15% | ギフト需要＋視覚的強さ（IG向き） |
| ⑤カフェ卸の裏側 | 10% | 卸先候補へのリーチ |

### 具体ネタ例（柱ごと）
- ①: 「同じ豆、焙煎度3段階で飲み比べ」「浅煎り→深煎りで味がこう変わる理由」「焙煎士が最初に見るのは○○」
- ②: 「エチオピア・イルガチェフェの華やかさはなぜ出るか」「ナチュラル精製 vs ウォッシュドの違い」
- ③: 「ハンドドリップで味がブレる3つの原因」「ペーパーフィルターの折り方で変わる」
- ④: 「ビンのラベルができるまで」「開封した瞬間の香りチャレンジ」
- ⑤: 「〇〇カフェさんに卸してる豆」「カフェのオーナーに聞いた選定基準」

## NG パターン

| NG | 理由 |
|---|---|
| 「コーヒーあるある」「コーヒー雑学」 | 通りすがりが集まるが、豆を買う層ではない |
| 「モーニングルーティン」系 | 差別化不能、競合だらけ |
| 「インスタント vs ドリップ」的な初心者解説 | ターゲット層には幼稚に映る |
| EC が未完成の段階での強い購買 CTA | 失望させる。まず「焙煎所フォローして待つ価値」を積む |
| 産地タグの使い回し（水の話で "産地" など） | すでに修正済み。エンリッチ由来の動的ラベルを使う |

## ファネル

```
SNS 視聴
  ↓ プロフィールへ移動
プロフィール bio（「焙煎士が毎日発信 / 豆のご購入はこちら」）
  ↓ EC URL タップ
自前 EC
  ↓ 初回購入
定期便 or 2 回目購入
  ↓ リピート
熱心なファン化（UGC投稿、紹介）
```

**EC 完成までの暫定 CTA**: 「EC 準備中です。フォローしてお待ちください」。
構築中は興味関心の貯金期間と割り切る。
