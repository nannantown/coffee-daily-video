# 戦略 — Open Ground Coffee

## 売りたいもの

**スペシャリティコーヒー豆の焙煎品**。自前EC（構築中）から直販。
カフェ卸も並行して継続（卸経由で知ってもらう入口にも機能）。

## 運用フォーマット: ニュース中心

**「毎朝チェックしないと乗り遅れる」FOMO を作る**ために、コンテンツはニュース性を主軸とする。エバーグリーン豆知識は単体では習慣化されにくい（いつ見てもいい = 見ない）。

| 曜日 | コンテンツ型 | 優先度 |
|---|---|---|
| 月〜金 | **毎朝のコーヒーニュース**（市況、産地、業界動向） | Primary |
| 土・日 | **エバーグリーン深掘り**（焙煎、産地、抽出） | Secondary |

週末エバーグリーンは「今週出たニュースの背景を深く知る」接続を意識する（ニュース駆動ニュース系情報として残る）。

## ペルソナ（優先順）

### Primary: 家で淹れる趣味層
- 既にコーヒー器具（ドリッパー、ミル）を所有
- 1 袋 ¥1,500〜¥3,000 に抵抗がない
- 産地・精製方法・ロースト日などの情報で買う豆を決める
- 失敗した淹れ方のリカバリに興味がある
- **この層を取れなければ事業成立しない**

### Secondary: 開業したての小規模カフェ店主
- 卸先として直接的な収益
- 1 店舗が決まると月5〜15kg の安定需要
- SNS は広告ではなく「焙煎所を知ってもらう信頼シグナル」として機能

### Tertiary: ギフト需要
- おしゃれなビン/パッケージが意思決定に効く
- 父の日、誕生日、開業祝いなど季節キャンペーンで拾う

## 差別化ポイント

1. **スペシャリティにこだわる知識量** — 焙煎士が一次情報を発信
2. **ビン・パッケージの意匠** — 贈答にそのまま使える
3. **SNS で焙煎士の顔が見える** — ECの店主が誰かわかる安心感

## コンテンツ柱（ニュース中心ver）

| 柱 | 比率 | 型 | ねらい |
|---|---|---|---|
| ①**市況ニュース** | 25% | News | C-price動向、為替、輸入原料価格の変化を人間語に翻訳 |
| ②**産地・ハーベストニュース** | 25% | News | 主要産地（エチオピア/ブラジル/コロンビア等）の収穫・気候ニュース |
| ③**業界・ビジネスニュース** | 20% | News | 大手/スタートアップ動向、サステナ、技術、カフェ業界 |
| ④**焙煎士の一次情報** | 15% | Evergreen | 既存の差別化軸、週末に寄せる |
| ⑤**OG入荷・裏側** | 15% | Product | 新入荷、ビン/パッケージ、卸エピソード（販売導線） |

### 具体ネタ例
- ①: 「アラビカ先物が〇%下落、豆価格への影響」「円安で輸入コストどう変わる」
- ②: 「エチオピアの今年の収穫、昨年比で何が変わった」「ブラジル干ばつの最新評価」
- ③: 「Blue Bottle の新戦略」「サステナ認証制度の最新動向」「某カフェチェーンが〇〇発表」
- ④: 「同じ豆、焙煎度3段階で飲み比べ」「浅煎り→深煎りで味がこう変わる理由」
- ⑤: 「今週入荷したエチオピア」「ビンラベルのデザイン誕生秘話」

## 探索方針（Discovery Strategy）

### 英語優先原則

**英語ソースを先に見る**。日本語メディアは英語を翻訳する形で1日遅れる。先に英語で拾い、日本語視聴者向けに翻訳・噛み砕く方が速報性・独自性・視聴者への価値で勝る。
日本語ソースは補完（英語で見つからないドメスティック情報や地域事情用）。

### 推奨ソース（シード）

**英語（優先）**:
- Perfect Daily Grind — 業界トピックの深堀り
- Daily Coffee News (Roast Magazine) — 業界ニュース
- SCA News (sca.coffee) — スペシャリティ協会公式
- Bloomberg / Reuters — 商品市況（Arabica Futures, Coffee C）
- World Coffee Research — 研究・品種
- Global Coffee Report — 国際動向
- Reddit r/Coffee, r/roasting — コミュニティ声

**日本語（バックアップ）**:
- SCAJ（日本スペシャルティコーヒー協会）公式発信
- 業界誌（カフェ＆レストラン、Coffee magazine JP）
- X（Twitter）日本語バリスタ界隈

### Discovery Methods（カテゴリタグ）

| タグ | 説明 |
|---|---|
| `news-en` | 英語の業界メディア・ニュースサイト |
| `news-ja` | 日本語の業界メディア |
| `market-data` | 商品市況、為替、指標データ |
| `official-src` | 産地機関、協会、WCR 等の公式発信 |
| `social-en` | Reddit / X（英語）のコミュニティ話題 |
| `social-ja` | X 日本語バリスタ・焙煎士界隈 |
| `trends` | Google Trends, X Trending |
| `expert-blog` | 業界有名人の個人発信・ブログ |

### 80/20 Exploit/Explore

- 過去14日で視聴率上位3位の method は 80% の頻度で使う（exploit）
- 残り 20% は**新しい method を試す**（explore）。探せば新カテゴリが見つかるはず
- Explore でヒットした method は次サイクルで exploit 側に昇格

### Meta PDCA（探し方の PDCA）

毎日のルーチンで以下を分析:

1. 直近 14 日の `performance-history.json` から **method → 平均 views** テーブルを作成
2. TOP 3 / WORST 3 method を特定
3. 「今日はどの method で探すか」を 80/20 ルールで決める
4. 新しい method のアイデアがあれば PDCA レポートの「Method 提案」セクションに記録
5. **WORST method は 2 週連続なら一旦凍結**（リストから外す、別 method 優先）

## NG パターン

| NG | 理由 |
|---|---|
| 「コーヒーあるある」「コーヒー雑学」 | 通りすがりが集まるが、豆を買う層ではない |
| 「モーニングルーティン」系 | 差別化不能、競合だらけ |
| 「インスタント vs ドリップ」的な初心者解説 | ターゲット層には幼稚に映る |
| EC が未完成の段階での強い購買 CTA | 失望させる。まず「焙煎所フォローして待つ価値」を積む |
| 産地タグの使い回し（水の話で "産地" など） | 動的ラベルを使う |
| **1週間以上前のニュースを新着扱い** | 鮮度が命。`freshness_hours > 168` は原則 NG |
| **出典不明ニュース** | 信用毀損。ソース URL を必ず記録・明示する |

## ファネル

```
SNS 視聴（今朝のコーヒーニュース）
  ↓ 「この人の情報が早くて深い」信頼蓄積
プロフィール bio（「焙煎士が毎日発信 / 豆のご購入はこちら」）
  ↓ EC URL タップ
自前 EC
  ↓ 初回購入（「入荷ニュース」で欲しくなる）
定期便 or 2 回目購入
  ↓ リピート
熱心なファン化（UGC投稿、紹介）
```

**EC 完成までの暫定 CTA**: 「EC 準備中です。フォローしてお待ちください」。
構築中は興味関心の貯金期間と割り切る。

## データスキーマ拡張（enriched-coffee-news.json）

ニュース駆動 + Discovery 記録のため、ルートレベルに `discovery` ブロックを追加:

```json
{
  "date": "2026-04-21",
  "discovery": {
    "method": "news-en",
    "description": "Perfect Daily Grind で今週のトップ記事を確認",
    "sources": ["https://perfectdailygrind.com/..."],
    "query": "arabica price april 2026",
    "freshness_hours": 14
  },
  "articles": [ ... ]
}
```

これを `record-upload.mjs` が `performance-history.json` に合流させ、後続の meta PDCA で使う。
